The Normal Distribution

Functionality:

This applet allows students to see how the Normal Distribution function changes as Standard Deviation and Mean are changed. 95% confidence intervals are shown and also shaded is the area conventionally tabulated in Normal Distribution tables.


Mean: S.Dev:

Remote Control using JavaScript

Both the Mean and Standard Deviation can be modified remotely from the applet. See the 'Runtime Applet API Calls' section below for details.

Examples

Set Mean = 19

Set S.Dev = 1.5

Set Mean = 21 and S.Dev = 2.3

Runtime Applet API Calls

This applet can be called into using Javascript while running. Although JavaScript can call the Applet directly, some convenient JavaScript functions have been created which deal with the details of the communication. The following table lists the call Names which can be made, and gives examples of their use and results.

API Call Name Parameters Use and Result
SetMean the new mean value javascript:qmn.setMean(20); javascript:qmn.setMeanSlider('qmMeanSlider'); Changes the mean value
SetStandev the new standard deviation javascript:qmn.setSD(2); javascript:qmn.setSDSlider('qmSDSlider'); Changes the standev value
showAbout none javascript:qmn.showAbout(); Shows the version dialog box of this applet.
